Use the Game Dev Hub MCP server.

1. Read AGENTS.md and list tasks with status ready_for_codex.
2. Pick the highest priority task.
3. Mark it in_progress.
4. Implement the smallest focused change in the Unity project.
5. Use unity_call_tool through the Dev Hub to inspect Unity, check console/compile status, and run relevant tests.
6. Post a result message with files changed, checks run, and remaining issues.
7. Mark the task codex_done.

Do not start unrelated features.
